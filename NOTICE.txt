This project uses a THUUyMorph-derived dictionary, based on the morphology-based Uyghur language segmentation corpus provided by Tsinghua University Natural Language Processing and Computational Social Science Lab.

More information and the dataset can be found at: http://thuuymorph.thunlp.org/

If you use this dataset in your work, please cite the following paper:
THUUyMorph-A Uyghur Morphological Analysis Corpus. In Proceedings of CCL/NLP-NABD 2017, Nanjing, October.

The dictionary file `thuuy_morph_raw.txt` bundled with this project is third-party data derived from THUUyMorph resources. The plugin derives original and split dictionary views from that source file.

The plugin code is licensed under the Apache License, Version 2.0. The THUUyMorph-derived dictionary data is documented separately here to preserve attribution and citation requirements for the original dataset.
