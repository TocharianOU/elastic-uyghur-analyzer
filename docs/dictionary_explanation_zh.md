# 词典说明

本文档说明 Elasticsearch 维吾尔语分析器插件中使用的词典系统，包括统一词典架构和形态学分析功能。

## 概述

插件使用**统一词典系统**，管理多个词典来源并提供词汇识别和形态还原功能。该系统包含预定义的形态学词汇库和用户自定义词汇。

## 词典架构

### 统一词典管理器

`UnifiedDictionaryManager` 类作为核心组件：
- 加载和解析多个词典来源
- 提供不同的分析视图（原始、分割、自定义）
- 管理基于优先级的词汇查找
- 支持完整词汇形式的识别和还原

### 词典文件位置

所有词典文件位于 `src/main/resources/dictionaries/`：

```
src/main/resources/dictionaries/
├── custom_dictionary.txt      # 用户自定义词汇（最高优先级）
└── thuuy_morph_raw.txt        # 清华形态学词汇库
```

## 词典格式解析

### THU词典格式规律

通过分析实际词典内容，发现以下格式规律：

#### 1. 基本词汇形式
```
ئا
ئائورتا
ئائىت
```
- 单独的词汇，无形态变化

#### 2. 元音弱化还原格式
```
ئائىلى.(ئائىلە) دىكى
ئورۇنلىش.(ئورۇنلاش) ىش
```
- 格式：`现代形式.(原始形式) 后缀组合`
- 括号内是元音弱化前的原始形式
- 点号后是该词汇的特定变位形式

#### 3. 简单后缀格式
```
ئائىلەر. گە
ئورۇن. نىڭ
```
- 格式：`词根. 后缀`
- 表示该词汇的特定语法形式

#### 4. 复杂后缀组合
```
ئائىلى.(ئائىلە) سى دىكى لەر نىڭ
ئورۇنلىش.(ئورۇنلاش) ىش قا
```
- 格式：`现代形式.(原始形式) 后缀1 后缀2 后缀3`
- 多个后缀的组合形式

#### 5. 复数变位标记
```
ئورۇن. لىر(لار) ى
ئوقۇغۇچى. لىر(لار) ى دىن
```
- `لىر(لار)` 表示复数形式的变体
- 根据元音和谐律选择适当形式

## 词典类型详解

### 1. 自定义词典 (`custom_dictionary.txt`)

**用途**：用户自定义的现代词汇，具有最高识别优先级。

**格式特点**：
```
# 基本形式
كومپيۇتېر
ئىنتېرنەت

# 变位形式
كومپيۇتېر. لار
تاكسى. دا
دوختۇر. نىڭ
```

**内容分类**：
- **现代科技词汇**：`كومپيۇتېر`、`ئىنتېرنەت`、`پروگرامما`
- **日常生活词汇**：`تاكسى`、`ئوتوبۇس`、`مېترو`
- **专业术语**：`دوختۇر`、`ئاسپىرانت`、`دىپلوم`

### 2. 清华形态学词汇库 (`thuuy_morph_raw.txt`)

**用途**：来自清华大学的综合维吾尔语形态学词汇数据集。

**数据规模**：约21,000条词汇形式

**格式解析**：
```
# 示例1：元音弱化还原
يېزىش.(يازىش) قا
# 现代形式: يېزىش
# 原始形式: يازىش
# 后缀: قا

# 示例2：复杂后缀组合
ئائىلى.(ئائىلە) سى دىكى لەر نىڭ
# 现代形式: ئائىلى
# 原始形式: ئائىلە
# 后缀组合: سى + دىكى + لەر + نىڭ

# 示例3：复数变位
ئورۇن. لىر(لار) ى نىڭ
# 词根: ئورۇن
# 复数标记: لىر/لار (根据元音和谐律)
# 后缀: ى + نىڭ
```

## 词典工作原理

### 词汇识别过程

系统**不是**进行分词，而是进行**词汇识别和形态还原**：

1. **完整匹配**：查找输入词汇是否在词典中存在
2. **形态还原**：如果找到，返回其基本形式和语法信息
3. **优先级处理**：自定义词典 > THU精确匹配 > THU部分匹配

### 实际使用示例

**输入词汇**：`ئورۇنلاشتۇرۇشلارنى`

**查找过程**：
1. 自定义词典查找：未找到
2. THU完整匹配：未找到完整词汇
3. THU部分匹配：找到相关条目
   - `ئورۇنلاش. قان لار`
   - `ئورۇنلىش.(ئورۇنلاش) ىش`
4. **分析结果**：识别为复合词汇的变位形式

## 分析视图说明

### 1. 原始视图 (Original View)
- **功能**：提供元音弱化还原后的原始形式
- **示例**：`يېزىش` → `يازىش`
- **用途**：学术研究、历史语言学分析

### 2. 分割视图 (Split View)
- **功能**：保持现代书写形式
- **示例**：`يېزىش` → `يېزىش`
- **用途**：现代文本处理、搜索应用

### 3. 自定义视图 (Custom View)
- **功能**：优先识别用户定义的现代词汇
- **示例**：`تاكسى` → 直接识别，无需形态分析
- **用途**：特定领域应用、专业术语处理

## 词典管理操作

### 添加自定义词汇

1. **编辑自定义词典**：
   ```bash
   vim src/main/resources/dictionaries/custom_dictionary.txt
   ```

2. **添加新词汇**：
   ```
   # 基本形式
   جديد_كلمة

   # 常用变位形式
   جديد_كلمة. لار
   جديد_كلمة. نىڭ
   جديد_كلمة. غا
   ```

3. **重新构建**：
   ```bash
   ./gradlew clean check
   ```

- **总识别能力**：涵盖古典和现代维吾尔语词汇

## 技术实现细节

### 词典加载流程

1. **文件读取**：读取所有词典文件
2. **格式解析**：
   - 识别括号内的原始形式
   - 解析点号后的后缀组合
   - 处理复数变位标记 `لىر(لار)`
3. **索引构建**：创建高效的查找表
4. **视图生成**：根据需要生成不同视图

### 解析算法

```java
// THU格式解析示例
if (line.contains(".(") && line.contains(")")) {
    // 格式: 现代形式.(原始形式) 后缀
    String modern = extractModernForm(line);
    String original = extractOriginalForm(line);
    String suffixes = extractSuffixes(line);
} else if (line.contains(". ")) {
    // 格式: 词根. 后缀
    String root = extractRoot(line);
    String suffixes = extractSuffixes(line);
} else {
    // 基本形式
    String word = line.trim();
}
```
